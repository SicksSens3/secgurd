# Secgurd

**DFIR toolkit for remote machine analysis**

---

## Overview

Secgurd is a lightweight Digital Forensics and Incident Response (DFIR) toolkit designed to simplify remote system analysis. It provides security teams and analysts with fast, actionable visibility into potentially compromised machines without requiring direct physical access.

---

## Why “Secgurd”?

Sigurd is a legendary hero in Norse and Germanic mythology, known for slaying the dragon Fafnir and facing overwhelming threats with precision and courage.

Secgurd carries that spirit forward—built to help analysts hunt, detect, and eliminate malicious activity in modern environments.
